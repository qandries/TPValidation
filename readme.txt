Ceci est un script pour installer virtualbox et vagrant.

Tutoriel:

1. Ecrire bash tpvalidationscript.sh
2. Appuyer sur 1 pour installer automatiquement la box du vagrantfile
3. Vagrant se lance
4. Quand il vous demande d'arrêter vagrant, appuyer sur y pour l'arrêter ou n si vous ne le voulez pas.

Erreur:
Il installe vagrant même si il est dèja installé.
